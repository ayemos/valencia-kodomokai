# バレンシア子供会 ウェブサイトリニューアルプロジェクト

## プロジェクト概要
バレンシア子供会のウェブサイトをモダンでモバイルフレンドリーなデザインにリニューアルします。シンプルなHTML/CSSのみで構築し、静的ホスティングサービスで簡単に運用できる構成を目指します。

## プロジェクト目標
- 📱 モバイルファーストのレスポンシブデザイン
- 🎨 既存のトンマナを維持しつつ、モダンで親しみやすいデザイン
- 🚀 HTMLとCSSのみのシンプルな構成
- 🔗 外部サービス（SNS、Google Form）との連携
- 📸 新しい写真素材を活用したビジュアル改善

## サイト構成

### ページ構成
```
/
├── index.html          # トップページ
├── about.html          # バレンシア子供会について
├── activities.html     # 活動内容
├── gallery.html        # フォトギャラリー
├── join.html          # 入会案内（Google Formへの誘導）
└── contact.html       # お問い合わせ
```

### アセット構成
```
/assets
├── css/
│   └── style.css      # メインスタイルシート
├── images/
│   ├── logo/          # ロゴ画像
│   ├── gallery/       # ギャラリー用写真
│   └── common/        # 共通画像
└── fonts/             # Webフォント（必要に応じて）
```

## デザインコンセプト

### カラーパレット
- プライマリカラー: 親しみやすい暖色系
- セカンダリカラー: 落ち着いたトーン
- アクセントカラー: 活動的な印象を与える色

### タイポグラフィ
- 見出し: 読みやすく親しみやすいフォント
- 本文: 可読性の高いシンプルなフォント
- 日本語Webフォントの使用を検討

### レイアウト
- モバイルファーストのレスポンシブデザイン
- シンプルで直感的なナビゲーション
- 余白を効果的に使った見やすいレイアウト

## 実装方針

### HTML
- セマンティックHTML5を使用
- アクセシビリティを考慮したマークアップ
- SEO最適化のための適切なメタタグ設定

### CSS
- CSSカスタムプロパティ（CSS変数）を活用
- Flexbox/Gridによる柔軟なレイアウト
- メディアクエリによるレスポンシブ対応
- アニメーションは控えめに使用

### 外部サービス連携
- **お知らせ**: SNS（Twitter/Instagram）のタイムライン埋め込み
- **入会申込**: Google Formへの誘導ボタン
- **お問い合わせ**: メールリンクまたはGoogle Form

## 開発環境

### 必要なツール
- テキストエディタ（VS Code推奨）
- ローカルサーバー（Live Server拡張機能など）
- モダンブラウザ（Chrome、Firefox、Safari）

### 開発手順
1. このリポジトリをクローン
2. ローカルサーバーを起動
3. ブラウザで確認しながら開発
4. 完成後、静的ホスティングサービスにアップロード

## デプロイ

### 推奨ホスティングサービス
- GitHub Pages（無料、簡単）
- Netlify（無料プラン有り、自動デプロイ）
- Vercel（無料プラン有り、高速）
- さくらのレンタルサーバー（有料、国内サービス）

### デプロイ手順
1. 全ファイルをホスティングサービスにアップロード
2. ドメイン設定（必要に応じて）
3. SSL証明書の設定（HTTPS化）
4. 動作確認
